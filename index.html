My Projects

Pet Adoption

This project is a full-stack web application designed to help users find pets available for adoption. It includes features such as searching for pets by type, breed, and location. The application is built using Python for the backend with Flask, and HTML, CSS, and JavaScript for the frontend. It integrates with a pet adoption API to provide up-to-date listings and includes a user authentication system for added functionality.

Bus Booking App

This project is a full-stack web application designed to help users to buy the products on online. It includes features such as searching for gadgets, watches, shoes. The application is built using the MERN stack: MongoDB for the database, Express.js and Node.js for the backend, and React.js for the frontend. It provides a seamless user experience with real-time updates and secure transactions.
